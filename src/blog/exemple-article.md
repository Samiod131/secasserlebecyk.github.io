## Formules Mathématiques

### Formule en ligne

Voici une formule inline : $E = mc^2$ qui s'intègre dans le texte.

### Formule en bloc

Voici une formule plus complexe :

$$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$

### Équations multiples

$$
\begin{aligned}
a^2 + b^2 &= c^2 \\
x &= \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
\end{aligned}
$$

## Code

Vous pouvez aussi inclure du code :

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
